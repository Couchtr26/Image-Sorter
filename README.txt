# Image Sorter - Codex Libris Mechanicus Visual Module

## 🔧 Description

A simple Python-based tool to automatically sort and rename image files into categorized folders by extension.

This tool allows users to:

- Automatically sort images into folders based on file type.
- Optionally rename images in batch using prefix and index sequencing.
- Easily select folders through a simple GUI interface.
- Supports multiple popular image formats.

Built to help photographers, designers, archivists, and hobbyists maintain organized image libraries with minimal effort.

---

## 💻 Features

- GUI interface (built with `tkinter`).
- Automatically sorts images into subfolders by extension.
- Supports optional batch renaming (prefix + 4-digit index).
- Lightweight and fully self-contained.

---

## 🛠 Supported Platforms

- Windows, macOS, Linux (requires Python 3.x installed)
